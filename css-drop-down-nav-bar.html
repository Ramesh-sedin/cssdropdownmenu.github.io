Even though many of us are still disinclined to start using CSS3 due to the creating interspire drop down menus of support in some browsers, there are those yui tabs in a dropdown menu there that are moving forward and doing some amazing stuff with its menu drop down zoo new features. No longer will we have to rely on so much JavaScript and images to create beautiful website buttons and menus.

You can create a horizontal menu drop down no wrap rounded navmenu with gradients, with no images and no Javascript. This javascript menu with css works perfectly well with Firefox, Opera, Chrome and Safari. The sharepoint cssmenu scroll down also works on non-CSS3 compitable browsers such as IE7+, but the css mouseover horizontal menu corners and shadow will not be rendered. CSS3 transitions could one day replace all the css menus with divider and submenus javascript animation things.
