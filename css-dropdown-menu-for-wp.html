No longer will we have to rely on so much JavaScript and images to create beautiful website buttons and menus.

You can create a vertical multi level flyout menu css rounded navmenu with gradients, with no images and no Javascript. This css attractive drop down navigation bar works perfectly well with Firefox, Opera, Chrome and Safari. The css3 collapsible vertical menu down also works on non-CSS3 compitable browsers such as IE7+, but the template on dropdown menu using css corners and shadow will not be rendered.
